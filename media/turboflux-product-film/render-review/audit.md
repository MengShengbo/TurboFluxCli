## 已关闭问题

- `[P1] 00:11.80 - 产品外壳重复入场`：后续 TUI 镜头曾重复执行 0.965→1 的缩放，播放时会产生轻微呼吸感。现仅首次产品展开与 45.5 秒产品重新进入时使用入场，连续工作段保持窗口几何稳定。
- `[P1] 00:13.00 - FastContext 右栏裁切`：检索镜头的全窗放大曾让 CURRENT TASK 状态靠近画面边缘。现取消外壳整体放大，检索证据改由内容层级完成聚焦。
- `[P1] 00:24.50 - 任务进度覆盖文本`：右栏进度条曾以绝对定位压住 Steps。现进度成为 rail 正常文档流的一部分，文本与进度在所有镜头保持稳定间距。
- `[P1] 全片 - 配乐响度不足`：首版预览为 -26.4 LUFS。生成器现通过 loudnorm 输出 -15.8 LUFS，真峰值保持在 -1.7 dBFS。
- `[P2] 00:29.00 - 验证镜头跳入近景`：Diff 镜头曾在切点直接出现 1.04 倍缩放。现由 0.9 秒 cameraGlide 推入，再横移到测试结果。

## 质量门槛

- MotionSpec lint：通过，0 错误，0 警告

- TypeScript：`tsc --noEmit` 通过

## 已知取舍

- 影片无旁白，依赖中文字幕与产品 UI 完成叙事，适合官网、社交媒体和会场静音首播。
- 配乐 LRA 为 1.9 LU，属于克制的环境电子底色；提示音承担动态强调，避免与高密度 UI 阅读竞争。
- 模型列表是 API 发现结果的演示状态，不构成默认模型或固定模型目录承诺。
